## Go Philosophy

### Go is Simple

The Go is designed to be simple and easy to understand.

It is similar to Python in terms of readability.

### Go is Efficient

Go is designed to be performant and efficient.

It is similar to C and C++ in terms of performance.

But unlike C and C++, Go has a garbage collector (GC). 

GC manages memory automatically. It frees up memory when it is no longer needed.

### Go is Concurrent

Go has native support for concurrency.

Instead of threads, Go uses goroutines which are lightweight version of threads.

### Go is Robust and Reliable

Go has a comprehensive standard library.

It minimizes the 3rd party dependencies.
