## Let's Build a Project - Part 1

You will build a `Grolyze` project.

It is a CLI Word Analytics Tool.

In this topic, you will apply what you learned so far:
- how to print to terminal
- structure of a Go program
- comments

### Create a new Go project

↳ 1) Make sure you are outside of this guide

Following command will push you to filesystem's root.

Feel free to choose a different directory.

```bash
cd /
```

↳ 2) Create a new directory for the project
```bash
mkdir grolyze
```

↳ 3) Navigate to the project directory
```bash
cd grolyze
```

↳ 4) Create a main.go file
```bash
touch main.go
``` 

↳ 5) Open the main.go file in your editor
```bash
code main.go
```

↳ 6) Write a program that prints "Grolyze: Guide Analyzer Initialized" to the terminal.

### Run the Program

You are building a project outside of this guide.

So, there is no command like `go run topic.go project` in your project.

Do you remember how to run your program ?

Use the following command:
```bash
go run main.go
```

✅ If everything is ok, you should see:
```
Grolyze: Guide Analyzer Initialized
```

The first step of building the project may be a bit easy.

But each chapter you will build more fun and challenging features.

🎉 Congratulations! You have setup your first project in Go.
